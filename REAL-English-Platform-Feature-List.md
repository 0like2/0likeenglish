## 🎓 학습 효과 측정 지표

### 대시보드에 표시할 지표
- **이번주 숙제 완성률** = (제출한 숙제 수 / 요구 숙제 수) × 100%
- **매일 숙제 완성률** = (완벽 + 중간) / 전체 과제
- **모의고사 평균 점수** = 전체 모의고사 점수 / 시험 횟수
- **월별 성적 변화** = 그래프 (점수 추이)
- **수업 진행도** = 완료한 수업 일지 수 / 전체 예약된 수업 수
